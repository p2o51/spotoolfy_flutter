# Spotoolfy Flutter

An MD3 Spotify API playback client with lyrics, note-taking, AI insights, and time machine features.

Search for lyrics and have them translated by Gemini, record your ratings and thoughts, chat with AI about your music, and rediscover songs you saved years ago on this day.

All in Material Design 3 style.
