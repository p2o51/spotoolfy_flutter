# 更新日志 (What's New)

## 2025年11月
*   **11-19**: 体验优化：预测性返回手势更加流畅，本地化功能增强。
*   **11-07**: Web 平台支持：接入 Worker 模拟设备，优化歌词显示样式和字体大小。
*   **11-06**: 翻译功能增强：新增海报歌词翻译支持，并为歌词翻译增加错误码。

*   **10-26**: 新功能：增加了分享页面（Beta）。

*   **06-19**: 本地化：增强本地化支持。

*   **04-15**: 多语言：新增中、英、日多语言支持。
